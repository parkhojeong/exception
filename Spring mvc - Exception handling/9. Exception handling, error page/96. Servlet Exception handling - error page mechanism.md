Exception handling flow
- controller throw Exception(or response.sendError) delivery to WAS
  - WAS <- filter <- servlet <- interceptor <- controller
- WAS find matched error handler and forward to it
  - WAS -> filter -> servlet -> interceptor -> controller(`/error-page/500`)
  - new ErrorPage(RuntimeException.class, "/error-page/500")
  - error information set request attribute
    - request attribute name
    - jakarta.servlet.error.
      - exception
      - exception_type
      - message
      - request_uri
      - servlet_name
      - status_code
