ExceptionResolver
- role: resolve Exception
- DispatcherServlet action after ExceptionResolver return
  - return empty ModelAndView() -> skip render and deliver ModelAndView Servlet to WAS
  - return ModelAndView with View -> render View
  - return null(all ExceptionResolver) -> DispatcherServlet throw exception and deliver to WAS 

usage
- Exception status code conversion
  - response.sendError(xxx) -> skip render and deliver error info to WAS
- view template
  - render error page by ModelAndView
- API response 
  - response.getWriter().write() -> skip render and deliver error info to WAS

* caution
- if use configureHandlerExceptionResolvers, remove ExceptionResolver
- instead, use extendHandlerExceptionResolvers
